# Chinook Music Store Data Analysis

## 📌 Project Overview

This project analyzes the **Chinook Music Store database** to understand customer behavior, sales performance, and music trends.
The analysis is performed using **SQL queries** to extract meaningful insights that can help the company improve its sales strategy.

## 🎯 Objective

The goal of this project is to analyze music sales data and answer important business questions such as:

* Which customers generate the highest revenue?
* Which genres and artists are most popular?
* Which countries generate the most sales?
* What are the top-selling tracks and albums?

## 🛠 Tools & Technologies

* **SQL (MySQL)**
* **Chinook Database**
* **Data Analysis Techniques**

  * JOIN
  * GROUP BY
  * Subqueries
  * CTE (Common Table Expressions)
  * Window Functions
  * Aggregate Functions

## 📂 Dataset

The dataset used in this project is the **Chinook Music Store database**, which contains information about:

* Customers
* Invoices
* Tracks
* Albums
* Artists
* Genres
* Employees

## 📊 Key Analysis Performed

Some of the important SQL queries used in this project include:

* Total revenue generated by each country
* Top customers based on spending
* Most popular music genres
* Best selling tracks
* Customer purchase patterns
* Sales trends analysis

## 📈 Insights

From the analysis, we discovered:

* Certain countries generate significantly higher revenue.
* A small number of customers contribute to a large portion of sales.
* Specific genres dominate the sales across multiple regions.

## 🚀 How to Run the Project

1. Import the **Chinook database** into MySQL.
2. Run the SQL queries provided in the project files.
3. Analyze the results to generate insights.
